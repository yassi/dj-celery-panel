# Django Celery Panel

Your Celery task inspector inside the Django admin.

## Overview

Django Celery Panel is a Django admin extension that provides a web interface for inspecting and managing your Celery tasks and workers.

**Status:** This project is currently in early development.

## Requirements

- Python 3.9+
- Django 4.2+
- Celery 5.0+
